Computer History of the 1970s

  • Directory

  1. Forth

    2. Forth is a imperative stack-based programming language
    that was designed and developed by Charles H. Moore.
    Forth was meant to be named Fourth for 4th generation
    software, which would have been the next generation,
    except the operating system they were working with
    limited file names to 5 characters.

  2. First Dot Matrix Printer

    3. Centronic introduces the first dot matrix printer, which
    prints by striking the paper with an ink soaked ribbon cloth.

  3. System/370

    4. IBM produced the first computer to be able to use virtual
    memory and memory chips instead of magnetic core
    technology. This computer was known as the System/370.

  4. Sealed Lead Acid Battery

    5. Sealed Lead Acid Batteries started being used in commercial use.

  5. VCR

    6. Phillips releases the first VCR during this time. VCR stood for
    video cassette recorder and could play or record a TV show to VHS.
